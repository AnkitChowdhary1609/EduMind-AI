An advanced Machine Learning and AI-powered academic analytics platform developed using **Python, Streamlit, Scikit-Learn, and AI models**.

The platform helps educational institutions analyze student performance, predict academic risks, perform clustering analysis, and generate AI-based recommendations using modern Machine Learning techniques.

# 📈 Machine Learning Models

## 🔹 Simple Linear Regression
Used to predict academic scores using one independent variable.

### Example
Predicting math scores using study hours.

---

## 🔹 Multiple Linear Regression
Uses multiple features together for prediction.

### Example
Using reading score, writing score, and attendance for prediction.

---

## 🔹 Logistic Regression
Used for classification tasks such as pass/fail prediction.

---

## 🔹 Naive Bayes
Used for sentiment analysis and probability-based classification.

---

## 🔹 K-Means Clustering
Groups students based on academic similarities.

---

## 🔹 PCA (Principal Component Analysis)
Reduces dimensions for better data visualization.
